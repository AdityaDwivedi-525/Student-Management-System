# Student Management System

A full-stack CRUD web application built using Node.js, Express.js, MongoDB, Mongoose, EJS, and Bootstrap. This application allows users to manage student records efficiently through a clean and responsive interface.

## Features

* Add new students
* View all student records
* Edit existing student details
* Delete student records
* Responsive card-based UI
* Form validation
* MongoDB database integration
* RESTful routes
* Method Override for PUT and DELETE requests

## Tech Stack

### Frontend

* HTML5
* CSS3
* Bootstrap 5
* EJS

### Backend

* Node.js
* Express.js

### Database

* MongoDB
* Mongoose

## Student Information Stored

* Name
* Email
* Age
* Gender
* Course
* Semester
* Address
* Phone Number

## CRUD Operations

* Create Student
* Read Student Records
* Update Student Information
* Delete Student Records

## Installation

```bash
git clone <repository-url>
cd student-management-system
npm install
```

Start MongoDB and run:

```bash
node app.js
```

Open in browser:

```text
http://localhost:8080/students
```
